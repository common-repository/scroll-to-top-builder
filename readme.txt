Scroll To Top Builder plugin allows the visitor to easily scroll back to the top of the page.

== Description ==

Scroll To Top Builder plugin allows the visitor to easily scroll back to the top of the page, with fully customizable options.

Scroll To Top Builder plugin has the following features.

### Features
* Animation speed
* Show after
* Scrolling Animation behavior
* Enable/Disable functionality
* Scroll to top text type
* Scroll to top icon type
* Icon font size
* Button Z-index
* Select User Devices (new)
* Select User Countries (new)
* Select  User Status (new)
* Custom CSS
* Custom JS
